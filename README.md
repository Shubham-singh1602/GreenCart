# GreenCart – Organic Grocery E-commerce

GreenCart is a full-stack e-commerce platform for organic grocery shopping.

## Tech Stack

Frontend
- React
- Axios
- CSS

Backend
- Spring Boot
- Hibernate
- MySQL
- REST APIs

## Features

- Product catalog
- Shopping cart
- User authentication
- Product browsing
- Checkout integration


## Installation

Backend

cd Server
mvn spring-boot:run

Frontend

cd Client
npm install
npm start
